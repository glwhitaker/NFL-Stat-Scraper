# NFL-Stat-Scraper
Python web scraper to pull statistics from each week.

## Weekly Overview
- writes to csv file:
  - Date of each game
  - Name of each team
  - Score of each team
  - Highest overall score of the week
  - Larget margin of victory of the week
 
## Team Overview
- allows the user to get an in depth look at any team
- writes to csv file:
  - team name
  - (more coming)
